# 科普中国全网涉科谣言智能监测预警系统建设项目

## 项目申报书

---

### 一、项目名称

科普中国全网涉科谣言智能监测预警系统建设

### 二、立项依据与必要性

#### 2.1 政策背景

2023年6月，科学辟谣平台受中央网信办邀请加入全国网络辟谣联动机制，标志着科学领域辟谣工作进入国家治理体系。2024年中央网信办开展"清朗"系列专项行动，将科技领域虚假信息列为重点整治对象。2025年《科技工作者科普创作指南》印发，进一步强调科普内容质量与科学传播规范。

科学辟谣工作已从单点辟谣进入系统治理阶段，需要从被动应对转向主动预警。

#### 2.2 现实基础与存在问题

科学辟谣平台自2019年建设以来，已取得显著成效：

- 总用户数超799万，累计传播量超91亿人次
- 覆盖50家全国学会的2965名专家
- 累计发布6680条权威辟谣内容
- 联合发布全网科学垂直领域辟谣榜单128期
- 内容累计登上全网热搜307个
- 连续六年获评中国互联网辟谣优秀作品

但在实际运营中，暴露出三个结构性问题：

**一是谣言源头感知滞后，前置发现能力不足。** 线索来源受"信息茧房"限制，对封闭社群、私域流量等高危传播场域监测存在盲区，难以捕捉苗头性风险，无法实现从事后处置向事前预警的有效前移。

**二是传播穿透力不强，精准引导实效受限。** 内容推送偏"大水漫灌"，对重点领域及老年等易感人群覆盖不足。算法推荐协同深度不够，权威声音难以有效触达核心受众。

**三是技术支撑体系薄弱，智能化治理能力待建。** 内部数据资源分散，缺乏数据驱动的研判与决策支持。工作高度依赖人工，AI等新技术应用停留在单点辅助，尚未形成"监测—研判—生产—评估"一体化的智能底座。

#### 2.3 涉科谣言的新形态与新威胁

当前涉科谣言已不再是简单的"信息真假"问题，而是呈现出三种具有系统性危害的新形态——**伪科普**：

**（一）伪科普→养老电诈。** 以"量子""石墨烯""负离子""干细胞""基因修复"等科学名词为包装，通过伪养生账号矩阵、社群裂变等方式，针对老年群体进行精准投放，最终导向保健品购买、理财投资等诈骗行为。本质上是借科学信任实施财产侵害。

**（二）伪科普→科技风评受损。** 针对核电、转基因、5G、人工智能、芯片自研、新能源等国家重大科技布局，通过断章取义、恐慌叙事、阴谋论等手法系统性抹黑，影响公众对科技强国战略的信心。部分内容存在境外信息操纵痕迹，与2035年科技强国目标直接冲突。

**（三）伪科普→邪教属性传播。** 以"能量疗愈""频率共振""量子意识""灵性觉醒"等伪科学概念为核心，构建封闭社群，形成精神依附和组织控制。具备反疫苗、反现代医学等特征，兼具经济剥削与精神控制的双重危害。

三种形态往往交叉出现：一个"量子能量水"产品可能同时涉及电诈（商品欺诈）、科技风评（歪曲量子力学）和邪教属性（社群封闭传播、精神依赖）。

#### 2.4 科普中国的独特优势

在所有可能承担此项工作的机构中，科普中国具有不可替代的优势：

- **科学判定权威性**：背靠50家全国学会的2965名专家，是唯一具备全学科覆盖的科学事实判定能力的平台
- **辟谣全链条经验**：六年运营积累了成熟的"发现—研判—创作—传播"机制
- **数据基础**：6680条辟谣内容形成的谣言库、辟谣库、专家库，是天然的训练数据和知识图谱基础
- **渠道联动能力**：全国网络辟谣联动机制成员，与网信办举报中心、主流媒体、网络平台有成熟协作关系
- **政治定位**：中国科协直属，五部委联合打造，具备权威发布的政治合法性

建设全网涉科谣言智能监测预警系统，是科学辟谣平台从"辟谣工具"升级为"科学信息安全基础设施"的关键一步。

### 三、项目目标

#### 3.1 总体目标

建设覆盖主要网络平台的涉科谣言智能监测预警系统，实现对涉科谣言和伪科普内容的7×24小时自动发现、智能研判、传播追踪和分级预警，定期输出传播路径分析报告与易感人群画像，为科学辟谣工作提供数据驱动的决策支持。

#### 3.2 具体目标

1. **监测覆盖**：覆盖微博、微信公众号、抖音、B站、知乎、百度贴吧等不少于6个主流平台，实现涉科内容7×24小时持续监测
2. **智能识别**：建成涉科谣言AI识别模型，对涉科内容实现自动判定，准确率不低于85%，并能区分谣言性质（普通误传/伪科普/恶意造谣）及危害主线（电诈/风评/邪教属性）
3. **传播追踪**：实现跨平台传播路径还原，对重点谣言事件输出传播节点图谱和扩散时间线
4. **分级预警**：建立三级预警机制（提示/预警/紧急），实现实时推送
5. **定期报告**：每周输出监测周报，每月输出传播路径分析报告与易感人群画像更新
6. **知识沉淀**：建成涉科谣言知识图谱，覆盖健康医疗、食品安全、前沿科技、公共安全等重点领域

### 四、项目内容与技术方案

#### 4.1 系统总体架构

系统采用四层架构：

```
┌─────────────────────────────────────────────┐
│            第四层：预警与输出层               │
│   实时预警 · 定期报告 · 易感人群画像 · 看板   │
├─────────────────────────────────────────────┤
│            第三层：传播分析层                 │
│   跨平台溯源 · 传播路径还原 · 节点识别        │
├─────────────────────────────────────────────┤
│            第二层：智能研判层（核心）          │
│   科学主张提取 · 事实核查 · 主线分类 · 分级    │
│   ┌──────────────────────────────────┐      │
│   │  科学知识图谱 + 大语言模型推理    │      │
│   └──────────────────────────────────┘      │
├─────────────────────────────────────────────┤
│            第一层：内容采集层                 │
│   多平台数据接入 · 关键词监测 · 账号追踪      │
└─────────────────────────────────────────────┘
```

#### 4.2 第一层：内容采集

**采集策略**

- **关键词种子库**：基于平台现有6680条辟谣内容和谣言库，提取高频伪科普关键词，建立动态更新的种子词库。初始规模不少于2000个关键词组合
- **重点账号追踪**：对已识别的伪科普矩阵号建立持续追踪列表，监测其内容发布和账号变异
- **热点事件触发**：当科技热点事件发生时（如核电站事件、新疫苗上市），自动扩大相关领域监测频率和范围
- **公众举报接入**：对接现有科学辟谣平台的公众举报通道，纳入监测线索源

**平台覆盖（分期实施）**

| 阶段 | 平台 | 数据接入方式 |
|------|------|-------------|
| 一期 | 微博、微信公众号、B站、知乎 | 开放API + 第三方数据服务 |
| 二期 | 抖音、快手、小红书、百度贴吧、今日头条 | 第三方数据服务为主 |
| 远期 | 微信群/朋友圈等私域 | 间接推断（公域异常指标反推私域传播） |

#### 4.3 第二层：智能研判（核心模块）

这是本系统区别于一般舆情监测产品的**核心能力**。市面上的舆情系统能做情感分析和热度追踪，但不具备判断"一条科普内容是否违反科学共识"的能力——这需要领域知识与AI推理的深度结合。

**研判流程**

```
采集内容（文本/视频转文字/图文OCR）
    ↓
涉科预筛选：是否涉及科学话题？
    ↓ 是
科学主张提取：该内容做了什么科学断言？
    例："石墨烯内裤能治前列腺炎"
    → 结构化主张：石墨烯材料具有治疗前列腺炎的效果
    ↓
事实核查：该主张是否有科学依据？
    → 知识图谱检索 + 大语言模型推理
    → 结论：无临床证据支持
    ↓
性质判定：
    → 普通误传 / 认知误区 / 伪科普（有组织有目的）/ 恶意造谣
    ↓
主线分类（伪科普进一步分类）：
    → 关联商品链接、导购行为 → 电诈主线
    → 涉及国家重大科技项目负面解读 → 科技风评主线
    → 来源封闭社群、高度同质化话术 → 邪教属性主线
    ↓
严重等级判定：
    → 提示（单条低传播）/ 预警（规模扩散中）/ 紧急（已成公众事件）
```

**科学知识图谱建设**

不从零建设，优先对接现有权威数据源：

| 数据源 | 覆盖领域 | 接入方式 |
|--------|----------|----------|
| 科学辟谣平台谣言库/辟谣库 | 全领域（6680条） | 内部数据 |
| 科普中国科普内容库 | 全领域 | 内部数据 |
| PubMed摘要 | 健康医疗 | 开放API |
| 中国知网科普文献 | 全领域 | 机构订阅 |
| 国家标准全文公开系统 | 食品安全、电磁、辐射等 | 公开数据 |
| 全国学会专家共识/指南 | 各学科 | 协调获取 |

知识图谱的功能不是"替代专家"，而是为AI研判提供**事实锚点**——有据可查，不依赖模型的参数化记忆。

**谣言分级分类标准**

| 类别 | 判定标准 | 应对策略 |
|------|---------|---------|
| 认知误区 | 非恶意，知识陈旧或理解偏差 | 科普引导 |
| 冷饭热炒 | 已有辟谣内容的重复传播 | 辟谣库匹配→精准推送 |
| 新发谣言 | 无先例，需专家研判 | 匹配学科专家→专题研判 |
| 伪科普-电诈 | 有组织、有商业导向 | 预警 + 对接反诈 |
| 伪科普-风评 | 针对国家科技战略 | 预警 + 专项监测 + 舆情报告 |
| 伪科普-邪教属性 | 封闭社群、精神控制特征 | 紧急预警 + 对接相关部门 |
| 重大突发 | 社会影响面大、传播速度快 | 一事一议 + 应急响应 |

#### 4.4 第三层：传播路径分析

**可实现的分析能力**

- **同源内容追踪**：基于文本相似度和语义匹配，识别同一谣言在不同平台、不同账号的变体传播
- **传播时间线**：还原"首发→跨平台扩散→二次加工→长尾传播"全过程
- **关键节点识别**：在传播链中识别起到放大作用的关键账号（KOL/营销号/水军节点）
- **矩阵行为检测**：多账号同步发布高度相似内容的协同传播行为识别
- **扩散速度建模**：对传播速度进行量化，支持"即将爆发"的早期预警

**传播特征指标体系**

| 指标 | 说明 | 预警意义 |
|------|------|---------|
| 首发→N平台时间差 | 跨平台扩散速度 | 速度越快，组织化程度越高 |
| 节点集中度 | 传播由少数账号主导的程度 | 高集中度提示有组织行为 |
| 内容变异度 | 传播中内容被改编的程度 | 高变异提示已进入"民间创作"阶段 |
| 矩阵同步率 | 多账号同时段发布相似内容 | 高同步率提示水军/矩阵号 |
| 私域渗透指标 | 公域无热度但阅读量异常 | 提示微信群等私域传播 |

#### 4.5 第四层：预警与输出

**实时预警机制**

| 等级 | 触发条件 | 通知方式 | 响应要求 |
|------|---------|---------|---------|
| 提示 | 单条伪科普内容，低传播量 | 系统内通知 | 纳入监测列表 |
| 预警 | 伪科普内容规模传播中，或涉及敏感领域 | 即时消息推送 | 24小时内研判 |
| 紧急 | 已形成公众事件，或涉及电诈/邪教属性 | 电话+消息 | 立即启动应急 |

**定期报告体系**

- **周报**：本周涉科谣言 Top 20、新增伪科普账号/矩阵、各主线活跃度变化、预警事件处置情况
- **月报**：传播路径分析（典型案例深度拆解）、易感人群画像更新、趋势预判、平台治理建议
- **专题报告**：重大科技事件后的专项监测报告（如某食品安全事件后的谣言传播追踪）
- **年度白皮书**：全年涉科谣言态势总结，为政策制定提供依据

### 五、易感人群画像

#### 5.1 方法论

不是"谁看了这条内容"，而是**"什么样的人在主动传播和深度互动伪科普内容"**。

**数据来源**
- 转发/评论/点赞/收藏伪科普内容的用户公开信息
- 用户主动发布的伪科普内容
- 用户关注列表和互动网络

**画像维度**

| 维度 | 指标 |
|------|------|
| 人口特征 | 年龄段推断、地域分布、性别倾向 |
| 内容偏好 | 关注的账号类型和话题领域 |
| 传播角色 | 被动接收者 / 主动转发者 / 二次创作者 / 核心节点 |
| 易感类型 | 健康焦虑型 / 技术恐惧型 / 阴谋论倾向型 / 利益驱动型 |
| 转化风险 | 是否有购买行为、是否加入封闭社群 |

#### 5.2 三条主线的典型画像

**电诈主线易感群体**：50岁以上，关注养生/中医/家庭健康类账号为主，活跃时段集中在早晨，高频转发带商品链接内容，社群传播特征明显（家族群、老同学群）。

**科技风评主线易感群体**：画像更分散。一类是真实的技术恐惧人群（如核电站周边居民、转基因种植区域居民），一类是信息操纵的传播节点，需通过行为模式区分。

**邪教属性主线易感群体**：封闭社群内高频互动，发布内容高度同质化（使用"频率""振动""觉醒""宇宙能量"等特定话术体系），关注列表重合度极高，具有明显的组织化特征。

#### 5.3 画像应用

- 为辟谣内容的**精准推送**提供目标人群定位
- 为反诈部门的**重点保护**提供人群识别支持
- 为科普工作的**选题策划**提供需求洞察
- 为政策制定提供**人群规模和分布**的数据支撑

### 八、预期成果

#### 8.1 直接成果

1. 涉科谣言智能监测预警系统1套，覆盖不少于6个主流网络平台

4. 监测周报不少于40期，月度报告不少于10期
5. 传播路径分析专题报告不少于6份

### 九、项目组织与保障

#### 9.1 组织保障

项目由科普中国统筹实施，依托科学辟谣平台现有团队和基础设施。重大技术决策和学科争议由全国学会专家委员会研判。

- 系统部署符合等级保护要求
- 数据存储和传输加密处理

- 定期评估系统准确率，持续优化模型
