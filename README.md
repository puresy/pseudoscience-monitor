# 全网伪科普谣言智能监测系统

一个普通人因为对科普领域乱象的关切，而做的开源项目。

## 项目动机

刷社交媒体时，经常看到各种打着"科学"旗号的谣言在传播——量子养生、酸碱体质、食物相克……这些伪科普内容不仅误导公众认知，有些甚至会危害健康。

我希望用技术手段做一点事：**自动发现、识别、追踪这些伪科普内容的传播**。

## 项目概述

本系统通过关键词监测 + 规则引擎 + AI分析，对社交平台上的涉科谣言进行：

- 🔍 **自动发现** — 基于关键词库和动态扩展策略持续监测
- 🧠 **智能研判** — 多层规则引擎 + LLM辅助分析，区分正常科普与伪科普
- 📊 **可视化看板** — 风险分级、传播趋势、高频谣言一目了然
- 📈 **传播追踪** — 分析谣言扩散路径和易感人群特征

## MVP 运行

```bash
# 抓取辟谣数据构建知识库
python mvp/piyao_crawler.py

# 运行完整分析流水线
python mvp/run_pipeline.py --source all

# 打开看板查看结果
open mvp/dashboard.html
```

## 技术栈

- **采集层**: requests + BeautifulSoup + Playwright（动态页面）
- **分析层**: 规则引擎（关键词匹配/模式识别/辟谣识别/信源白名单）+ LLM辅助（小米 MiMo-V2-Pro）
- **展示层**: 原生 HTML/CSS/JS 看板（支持逐条明细核查）
- **自动化**: GitHub Actions（每日凌晨3点定时采集+分析）
