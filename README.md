#### 第一部分：关键词库设计 (5个词库)

| 词库类型 | 条目数 | 风险分数 | 应用场景 |
|--------|--------|--------|--------|
| 恐惧营销触发词库 | 50+ | 9/10 | 直接检测恐吓营销文本 |
| 伪权威虚构词库 | 60+ | 9/10 | 识别虚假研究/专家 |

| 因果简化触发词库 | 40+ | 7/10 | 识别错误因果逻辑 |
| 营销导向词库 | 50+ | 7/10 | 检测商业推广意图 |

**包含关键词示例**:
- 恐惧词: 致癌、有毒、隐形杀手、必须立即
- 虚假权威: 某研究发现、某大学、医生都推荐（无具体来源）
- 科学滥用: 量子芯片、纳米能量、干细胞唤醒
- 因果简化: 只有、就是、完全、必然导致
- 营销词: 代理招募、赚大钱、限时优惠、不用上班

#### 第二部分：识别规则引擎

**4层优先级系统**:
```
CRITICAL (极高风险) - 需立即处理
HIGH (高风险) - 需标记警告
MEDIUM (中风险) - 需提醒审视
LOW (低风险) - 常规科普
```

**5条关键规则** (R1-R5):
- R1: 恐惧营销模式 (多恐惧词+绝对词+链接)
- R2: 权威虚构+见证堆砌
- R3: 科学滥用+功效夸大
- R4: 因果简化+绝对化
- R5: 个案证据代替统计

**规则示例** (可直接用于代码实现):
```yaml
规则 R1: 恐惧营销模式
IF (恐惧词 > 2) 
AND (触发词 IN ["立即", "赶快", "别再等"])
AND (包含产品推荐 OR 包含链接)
THEN Risk = CRITICAL
置信度: 95%
```

#### 第三部分：领域分类关键词

**4个主要域**：
1. **健康保健** - 疾病名、症状词、保健品词
2. **食品安全** - 农药残留、添加剂、有毒物质、相克触发词
3. **美容护肤** - 效果承诺、成分宣传、绝对承诺
4. **科技产品** - 前沿技术词、性能词、虚假安全承诺

**检测流程** (4阶段):
1. 关键词匹配检测
2. 模式识别检测

#### 2. 实施检测规则
```python
# 参照规则R1-R5实现检测逻辑
if (fear_word_count > 2) and ("立即" in text) and has_product_link:
    risk_level = "CRITICAL"
```

#### 3. 触发风险评估
```python
# 按照4层优先级系统标记内容

    action = "标记警告"
```
