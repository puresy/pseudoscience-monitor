## 二、识别规则引擎

### 2.1 规则优先级体系

```yaml
级别: CRITICAL (极高风险)
  - 多个恐惧词 + 绝对化词 + 产品链接
  - 虚假权威 + 夸大效能 + 代理招募
  - 科学概念滥用 + "100%有效" + 购买按钮
  - 触发阈值: 同时满足≥2类规则

级别: HIGH (高风险)
  - 单恐惧词 + 解决方案词
  - 虚假权威 + 产品推荐
  - 科学术语 + 夸大效能
  - 因果简化 + 复杂疾病
  - 触发阈值: 满足任意1-2类规则

级别: MEDIUM (中风险)
  - 恐惧词 + 教科书知识
  - 片面对比 + 可查证信息
  - 科学术语 + 合理应用
  - 单个夸大词汇
  - 触发阈值: 包含≥1个风险特征但信息可验证

级别: LOW (低风险)
  - 正常科普内容
  - 无明显风险词汇
  - 包含具体来源和数据
```

### 2.2 组合规则示例

#### 规则 R1: 恐惧营销模式
```
IF (恐惧词 > 2) 
AND (触发词类 ∈ ["立即", "赶快", "别再等"])
AND (包含产品推荐 OR 包含链接)
THEN Risk = CRITICAL
置信度: 95%
```

**实际例子**:
```
"不知道吧！这种致癌物就在你家里！
立即清理，用XX产品彻底去除。
已有30万人购买，限时优惠！"
```

#### 规则 R2: 权威虚构 + 见证堆砌
```
IF (权威虚构词 ∈ [某研究,某大学,据说])
AND (无具体来源信息)
AND (见证词数量 > 3)
THEN Risk = HIGH
置信度: 90%
```

**实际例子**:
```
"某研究发现，我朋友用了效果特别好，
我妈妈也在吃，医生都推荐，
99%用户好评"
```

#### 规则 R3: 科学滥用 + 功效夸大
```
IF (科学术语 ∈ [量子,纳米,干细胞])
AND (无医学背景应用)
AND (夸大词 ∈ [100%,永久,彻底])
THEN Risk = CRITICAL
置信度: 98%
```

**实际例子**:
```
"量子纳米干细胞芯片，100%有效，
永久改善睡眠"
```

#### 规则 R4: 因果简化 + 绝对化
```
IF (单因素词 ∈ [只有,就是,完全])
AND (复杂结果 ∈ [癌症,糖尿病,心病])
AND (无多因素说明)
THEN Risk = HIGH
置信度: 92%
```

**实际例子**:
```
"脂肪肝就是由肥胖引起"（忽视其他因素）
❌ 正确表述：脂肪肝可由多种因素引起，
包括肥胖、不当减肥、酒精摄入等
```

#### 规则 R5: 个案证据代替统计
```
IF (个案见证词 > 2)
AND (无大样本数据)
AND (无对照组信息)
THEN Risk = MEDIUM
置信度: 85%
```

**实际例子**:
```
"我用了效果特别好，身边朋友都说好"
vs
"临床试验显示：308例患者中，
治疗组有效率78%，对照组23%"
```

### 2.3 多维度组合检测

#### 文本维度检测清单

| 检测项 | 触发阈值 | 权重 | 示例 |
|-------|--------|------|------|
| 恐惧词密度 | >5% | 0.3 | 每100词包含5+恐惧词 |
| 绝对词覆盖 | >3个 | 0.25 | 100%、永久、彻底、绝对 |
| 权威虚构 | 包含但无源 | 0.25 | "某研究发现"无论文号 |
| 产品链接 | 出现 | 0.2 | 购买、购物车、代理 |
| 见证堆砌 | >5个 | 0.15 | 我、朋友、用户、好评 |

**综合风险分数计算**:
```
Risk_Score = Σ(检测项权重 × 命中得分)
           + 组合规则加分

HIGH_RISK_THRESHOLD = 7.0/10
MEDIUM_RISK_THRESHOLD = 5.0/10
```

## 四、实施指南

### 4.1 检测流程

```
输入文本
  ↓
[阶段1] 关键词匹配检测
  - 恐惧词检测
  - 虚假权威检测
  - 科学滥用检测
  ↓
[阶段2] 模式识别检测
  - 规则R1-R5组合检测
  - 组合词组检测
  ↓
[阶段3] 上下文理解
  - 语义相关度检测
  - 论证逻辑检测
  ↓
[阶段4] 风险评分
  - 综合风险计算
  - 置信度评估
  ↓
输出: 风险等级 + 触发规则 + 建议处理方式
```

### 4.2 规则优化周期

- **周期**: 每月1次
- **数据源**: 科学辟谣网站月度流言榜
- **更新内容**:
  1. 新出现的谣言关键词
  2. 新的虚假权威声称
  3. 新的科学概念滥用
  4. 新的营销话术
- **反馈机制**: 错判案例 → 规则调整

### 4.3 黑名单库建设

**类型1: 虚假研究机构**
```
"某研究所发现"、"国际研究中心"、
"医学联合会"、"健康协会"等无法查证的机构
```

**类型2: 虚假专家**
```
使用"某医生"、"某教授"等模糊称谓，
无法通过医疗系统查证的个人
```

**类型3: 反复出现的虚假论述**
```
"某食物与某食物相克会中毒"
"XX病就由YY引起"
等反复被辟谣但持续传播的谣言
```

## 五、性能指标

### 5.1 目标准确率

| 指标 | 目标值 | 说明 |
|------|-------|------|
| 精准度 (Precision) | >92% | 标记的伪科普中真正是伪科普的比例 |
| 召回率 (Recall) | >85% | 全部伪科普中被正确标记的比例 |
| F1-Score | >88% | 精准度和召回率的综合指标 |
| 误判率 | <8% | 将正常科普错判为伪科普 |

### 5.2 测试验证集示例

**CRITICAL案例**:
```
"震撼！这个食物99%的人都吃错了，
会导致癌症！立即转发拯救家人，
现在购买排毒产品五折优惠！"
预期: CRITICAL ✓
```

**HIGH案例**:
```
"某研究发现，褪黑素能彻底治疗失眠，
用过的人都说特别有效。"
预期: HIGH ✓
```

**MEDIUM案例**:
```
"高温洗衣可能洗得更干净，
但也要考虑衣物材质。"
预期: MEDIUM ✓
```
